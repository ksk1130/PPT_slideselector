# PowerPoint スライド選択 アドイン インストール手順

## 1. 受け取るファイル

以下の2ファイルを同じフォルダに置いてください。

- [slideselector.pptm](slideselector.pptm)
- [install.bat](install.bat)

## 2. インストール

1. `install.bat` をダブルクリック
2. 黒い画面が出たら、そのまま終了するまで待つ
3. **PowerPoint を完全に終了**（全ウィンドウを閉じる）
4. PowerPoint を再起動

## 3. 使い方

1. リボンに **「スライド選択」** タブが表示されます
2. **「スライド選択」** ボタンをクリック
3. リストからスライド番号を選んでジャンプ
   - ダブルクリックでもジャンプできます

## 4. うまく表示されない場合

### リボンが出ない
- PowerPoint を完全終了して再起動
- それでも出ない場合は IT 部門に連絡

### マクロが無効になった
- 「コンテンツの有効化」をクリック
- それでも動かない場合は IT 部門に連絡

---

## 参考（インストール先）

このファイルは自動で以下にコピーされます：

```
C:\Users\(ユーザー名)\AppData\Roaming\Microsoft\AddIns\slideselector.pptm
```

# PowerPoint スライド選択 インストール手順

## 📋 インストール方法（ユーザー向け）

### ステップ 1: インストーラーを実行

1. IT部門から受け取った以下のファイルを確認：
   - `install.bat`（インストーラー）
   - `SlideJumper.pptm`（アドインファイル）

2. **`install.bat` をダブルクリック**
   
   ![インストーラー実行](https://via.placeholder.com/400x200?text=install.bat+をダブルクリック)

3. 黒いウィンドウが表示され、自動でインストール処理が進行します

4. 「インストール完了しました！」というメッセージが表示されたら、**任意のキーを押す**

### ステップ 2: PowerPoint を再起動

1. **PowerPoint を完全に終了する**
   - PowerPoint の全てのウィンドウを閉じる
   - タスクバーに PowerPoint がないことを確認

2. **PowerPoint を再起動する**
   - PowerPoint アイコンをダブルクリック

### ステップ 3: 確認

1. PowerPoint が起動したら、任意のプレゼンテーションを開く

2. **「ホーム」タブを確認**
   
   リボンに「スライド操作」というグループが表示され、その中に「スライド選択」ボタンが見えます

3. **「スライド選択」ボタンをクリック**
   
   スライド選択ダイアログが表示されれば、インストール成功です ✅

---

## 🚀 使用方法

### スライドにジャンプする

1. PowerPoint で任意のプレゼンテーションを開く

2. **リボンの「スライド操作」グループにある「スライド選択」ボタンをクリック**
   
   スライド一覧ダイアログが表示されます

3. **リストから目的のスライドを選択**
   
   - クリックで選択
   - または、ダブルクリックで即座にジャンプ

4. **「ジャンプ」ボタンをクリック**
   
   選択したスライドに移動します

## ❌ トラブルシューティング

### Q: 「スライド選択」ボタンが表示されない

**A:** 以下をお試しください：

1. **PowerPoint を完全に終了**
   - タスクマネージャーで PowerPoint プロセスがないことを確認

2. **PowerPoint を再起動**

3. それでも表示されない場合は、IT部門までお問い合わせください

### Q: install.bat を実行するとエラーが出た

**A:** 以下をご確認ください：

1. `install.bat` と `SlideJumper.pptm` が同じフォルダにあるか
2. ファイルが削除・移動されていないか
3. 管理者権限が必要な場合：
   - `install.bat` を右クリック
   - 「管理者として実行」を選択

### Q: アドインを削除したい

**A:** `uninstall.bat` をダブルクリックしてください
